Get started with Multi language editor

## Download Project
 
 ```bash
git clone https://github.com/11joselu/multi-language-editor.git
 ```
 
## Compile Project
 
 ```bash
 $ cd multi-language-editor
 ```
  
 ```maven
 $ mvn clean install
 ```
